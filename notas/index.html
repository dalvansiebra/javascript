Imagine vagas de estacionamento devidamente identificadas "a1, a2, a3, a4". Cada vaga representa uma varial. Para colocar um carro em uma vaga.

Regras para identificar variáveis:

  • *Podem começar com letra, $ ou _
  • *Não podem começar com números
  • *É possovel usar letras ou números
  • *É possivel usa acentos e símbolos
  • *Não podem conter estaços //Substituir por _
  • *Não podem ser palavras reservadas //Comandos do JS
  • *Letras maiúscilas e minpusculas fazem diferença
  • *Tente utilizar nomes coerentes para as variáveis, ex: nome = nome, idade = idade, telefone = telefone

TRATAMENTO DE DADOS

Conversão de string para number

  • var n1 = window.prompt('Digite um número.')
  • var n2 = Number(window.prompt('Digite outro número.'))

O comando window.prompt retorna uma string, precisamos converter em number

Podemos utilizar três formas antes do comando:

  • Number.parseInt(n) // Números inteiros
  • Number.parseFloat(n) //Números flutuantes/reais
  • Number // O JS identifica se é inteiro ou real

Conversão de number para string

  • String(n)
  • n.toString()

Formatando Strings

    var s = 'JavaScript'
  • 'Eu estou aprendendo s' // não faz interpolação
  • 'Eu estou aprendendo' + s // usa interpolação
  • `Eu estou aprendendo ${s}` // usa template string, tem que ser com cráse
  • $.length // quantos caracteres a string tem
  • $.toUpperCase() // tudo para 'MAIÚSCULAS'
  • $.toLowerCase() // tudo para 'minúsculas'

Formatando Numbers

  • n1.toFixed(2) // colocar duas casas decimais
  • n1.toFixed(2).replace('.', ',') // trocar o . pela ,
  • n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) // colocar o símbolo de Real
  • n1.toLocaleString('pt-BR', {style: 'currency', currency: 'USD'}) // colocar o símbolo de Dólar
  • n1.toLocaleString('pt-BR', {style: 'currency', currency: 'EUR'}) // colocar o símbolo de Euro

OPERADORES PARTE 01

    Tipos de Operadores
  • aritméticos
  • atribuição
  • relacionais
  • lógicos
  • ternário

Operadores aritimédticos são usados para fazer cálculo

  • + adição

    serve para adição, por ex: 5 + 2 = 7

  • - subtração

    serve para subtração, por ex: 5 - 2 = 3

  • * multiplicação

    serve para multiplicação, por ex: 5 * 2 = 10

  • / divisão real

    serve para divisão real, por ex: 5 / 2 = 2.5

  • % resto da divisão

    mostra o resto da divisão inteira, por ex: 5 % 2 = 1

  • ** potência

    serve para elevar a potência, por ex: 5 ** 2 = 25

Precedência de operadores

Por ex: 5 + 3 / 2 = 6.5

Ordem de precedência

  1. (); // o que tiver entre parênteses
  2. **; // potências
  3. * / %; // multiplicação, divisão, resto da divisão. Quem aparecer primeiro
  4. + -; // adição e subtração

Atribuições simples

    var n = 3
  • n = n + 4 resultado: 7
  • n = n - 5 resultado: 2
  • n = n * 4 resultado: 8
  • n = n / 2 resultado: 4
  • n = n ** 2 resultado: 16
  • n = n % 5 resultado: 1

Simplificando

    var n = 3
  • n = n + 4 resultado: 7 // agora fica n += 4
  • n = n - 5 resultado: 2 // agora fica n -= 4
  • n = n * 4 resultado: 8 // agora fica n *= 4
  • n = n / 2 resultado: 4 // agora fica n /= 4
  • n = n ** 2 resultado: 16 // agora fica n **= 4
  • n = n % 5 resultado: 1 // agora fica n %= 4

Operadores relacionais

O resultado de toda operação relacional será resultado boleano, true ou false

  • > maior que (5 > 2 true)
  • < menor que (7 < 4 false)
  • >= maio ou igual (8 >= 8 true)
  • <= menor ou igual (9 <= 7 false)
  • == igual (5 == 5 true)
  • != diferente (4 != 4 false)

Operadores de identidade, verifica se é do mesmo valor e do mesmo tipo

  • 5 == 5 true
  • 5 == '5' true
  • 5 === '5' false // estritamente igual
  • 5 === 5 true

Operadores Lógicos

  • ! negação/oposição
  • && conjunção/e
  • || fisjunção/ou
  • ! true = false
  • ! false = true
  • true && true = true
  • true && false = false
  • false && true = false
  • false && false = false
  • true || true = true
  • true || false = true
  • false || true = true
  • false || false = false

CONDIÇÕES

  • Condições Simples
  • Condições Compostas
  • Condições Alinhadas
  • Condições Múltiplas

Condição Simples

Contém apenas o if e não contém o else / senão

if (condição) {

console.log('Escreva o que quiser!')

var.innerHTML = 'Escreva o que quiser!'

}

Condição Composta

Contém o if e o senão

if (condição) {

console.log('Escreva o que quiser!')

} else {

var.innerHTML = 'Escreva o que quiser!'

}

Condição Aninhada

Contém o else if onde põe um im if dentro de outro if, criando várias condições.

if (condição) {

console.log('Escreva o que quiser!')

} else if (condição) {

var.innerHTML = 'Escreva o que quiser!'

} else if (condição) {

var.innerHTML = 'Escreva o que quiser!'

} else if (condição) {

var.innerHTML = 'Escreva o que quiser!'

} else {

var.innerHTML = 'Escreva o que quiser!'

}

Condição Múltipla

Serve para valores fixos, para situações específicas.

switch (expressão) {

case 0:

console.log('Domingo')

break

case 1:

console.log('Segunda')

break

case 2:

console.log('Terça')

break

case 3:

console.log('Quarta')

break

case 4:

console.log('Quinta')

break

case 5:

console.log('Sexta')

break

case 6:

console.log('Sábado')

break

default

console.log('ERRO dia inválido')

break

}
