VARIÁVEIS E TIPOS PRIMITIVOS

Dois tipos de colocar comentários em JS.
// Em uma única linha
/*
Em mais de uma linha
*/

Imagine vagas de estacionamento devidamente identificadas "a1, a2, a3, a4". Cada vaga representa uma varial. Para colocar um carro em uma vaga.

  • var a1 = carro1
  • var a2 = carro2
  • var a3 = null //Significa que a variável está nula
  • No JS o sinal de "=" significa "recebe".
  • Colcar valores dentro de variáveis significa "atribuição".
  • Para colocar texto, colocar ",' ou `.

Regras para identificar variáveis:

  • *Podem começar com letra, $ ou _
  • *Não podem começar com números
  • *É possovel usar letras ou números
  • *É possivel usa acentos e símbolos
  • *Não podem conter estaços //Substituir por _
  • *Não podem ser palavras reservadas //Comandos do JS
  • *Letras maiúscilas e minpusculas fazem diferença
  • *Tente utilizar nomes coerentes para as variáveis, ex: nome = nome, idade = idade, telefone = telefone

Aula 06

TRATAMENTO DE DADOS

Conversão de string para number

  • var n1 = window.prompt('Digite um número.')
  • var n2 = Number(window.prompt('Digite outro número.'))

O comando window.prompt retorna uma string, precisamos converter em number

Podemos utilizar três formas antes do comando:

  • Number.parseInt(n) // Números inteiros
  • Number.parseFloat(n) //Números flutuantes/reais
  • Number // O JS identifica se é inteiro ou real

Conversão de number para string

  • String(n)
  • n.toString()

Formatando Strings

    var s = 'JavaScript'
  • 'Eu estou aprendendo s' // não faz interpolação
  • 'Eu estou aprendendo' + s // usa interpolação
  • `Eu estou aprendendo ${s}` // usa template string, tem que ser com cráse
  • s.length // quantos caracteres a string tem
  • s.toUpperCase() // tudo para 'MAIÚSCULAS'
  • S.toLowerCase() // tudo para 'minúsculas'

Formatando Numbers

  • n1.toFixed(2) // colocar duas casas decimais
  • n1.toFixed(2).replace('.', ',') // trocar o . pela ,
  • n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) // colocar o símbolo de Real
  • n1.toLocaleString('pt-BR', {style: 'currency', currency: 'USD'}) // colocar o símbolo de Dólar
  • n1.toLocaleString('pt-BR', {style: 'currency', currency: 'EUR'}) // colocar o símbolo de Euro
